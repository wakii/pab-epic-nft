- To run this frontend app
```
cd packages/react-app
npm start
npm run start
```
---
- Smart contract file written in Solidity, deployed with hardhat
```
cd packages/hardhat/
npx hardhat compile
npx hardhat run scripts/run.js --network rinkeby   
```
